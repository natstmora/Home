This data was Provided by Nasdaq free Data published by Quandl. Visit RI for information about inflation calculation.

After downloading the zipped CSV the spreadsheet should look like this. The data in the description column needs to be gathered.

Then gather the information for the United States Consumer Price Index from Federal Reserve Economic Data.

Now right click the value 304 select inspect from the drop down menu. The Inspector will open with the HTML code selected. Then select Copy select the arrow drop down menu > Copy full XPath

Then go back to the spreadsheet get the link to the website and add it to the =IMPORTXML function. The link to the website will be added to the left and the XPath will be added to the right.

Then do this for the inflation rate for United States by getting the data for 10 years at Federal Reserve Economic Data.

Add the link to the spreadsheet using =IMPORTXML function.

The Federal Reserve Economic Data has data for the other countries. Another way to gather the data and add it to the spreadsheet is to go to the link listed in description column. For the completed spreadsheet go to GitHub.
